# Minions Scheduler

Schedules, cron triggers, and timed execution definitions

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-scheduler/sdk`) — Core library for Node.js
- **Python SDK** (`minions-scheduler`) — Python bindings
- **CLI Tool** (`@minions-scheduler/cli`) — Command-line interface
- **Web App** — Interactive playground at [scheduler.minions.wtf](https://scheduler.minions.wtf)
